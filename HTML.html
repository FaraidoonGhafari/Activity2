What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating web pages.

Example of a basic HTML structure:

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>

  <h1>This is a Heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>
